字典

字典的概念

字典是一种可变容器模型，类似于我们生活中使用的字典，它可以存储任意类型对象，并能够对存储的对象进行修改。与列表、集合不同的是，字典的每个元素都是由一个键和一个值组成的“键值对”，键和值通过冒号分开。
